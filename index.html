Miniatur Converter

Klik atau drop gambar di sini

Cara Pakai:

  1. Pilih tab Image to URL.
  2. Drop gambar ke box. Halaman otomatis refresh, nama file dan URL akan muncul.
  3. Pindah ke tab URL to Miniatur untuk membuat miniatur.
  4. Masukkan URL hasil upload dan klik URL to Miniatur.
  5. Download hasil miniatur.
